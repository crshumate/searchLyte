This jQuery plugin is a search-as-you-type plugin based off of Google's live search. It is fairly simple to use. The plugin needs to know about three items:

  • The Search List

    The search list is where you place your pre-populated search strings and urls. It is on the roadmap to make this plugin ajax compatible. Currently however you must pre-populate your search results and save it in a hidden div on the page. The default id for this div is #list but it is customizable via the options array. The search string and url must be saved in the following format:

    • Random Search String::http://facebook.com;

    Notice the search string and url is separated by a double colon, and the entire search string ends with a semicolon. It is imperative you follow this format else the plugin will not parse your search results correctly.

  • The Result Box

    The result box contains all relevant results for your current query and is updated as you type each character. When the results appear you can arrow down through the results and hit return and you will be redirected to the url associated with the search string.

    In order for the search box to automatically scroll as you reach the bottom of your current results list it makes use of jQuery.position(). In order for .position() to return the correct position your result box must be absolutely, relatively or fixed positioned. Static positioning returns the position as it relates to the first non-static parent. The searchLyte.css takes care of this for you and makes it absolutely positioned while wrapped in a relatively positioned wrapper. Simply copy the HTML below and attach the CSS and JS files.

    The result box by default has an id of searchLyte but this is customizable via the options array.

  • The Query Input

    The query input has no default id as this is where you attach the $.searchLyte plugin. An example of this would be:
    $("#my-cool-input").searchLyte();

Final Thoughts

To get you started simply copy and paste the code below into your page.

The Head of your Page

	
		<head>	
		    	<link type='text/css' rel='stylesheet' href='searchLyte.css'  />
			<script type="text/javascript" src="http://code.jquery.com/jquery-1.8.0.min.js"></script>
			<script type="text/javascript" src="jQuery.searchLyte.js"></script>
		</head>

The JS



		
	$("#my-cool-input").searchLyte({
		searchList:'#list',
        	resultBox:'#searchLyte',
		ajax:false,
		scrollSpeed:300, //how fast the results box scrolls through your results
		autoComplete: 'off' //sets the autoComplete attribute to off so it plays nice with your browser
	});
